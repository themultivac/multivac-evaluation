# Part 1: Rindler coordinates from constant proper acceleration

We work in \(1+1\) dimensions first, since the essential physics is in the \((t,x)\) plane. The transverse coordinates \(y,z\) are spectators and can be added at the end.

We use Minkowski metric with signature \((+,-)\):

\[
ds^2 = c^2 dt^2 - dx^2.
\]

---

## 1. Uniform proper acceleration in Minkowski spacetime

A worldline \(x^\mu(\tau) = (ct(\tau),x(\tau))\) parameterized by proper time \(\tau\) has four-velocity

\[
u^\mu = \frac{dx^\mu}{d\tau}
\]

with normalization

\[
u^\mu u_\mu = c^2.
\]

The four-acceleration is

\[
a^\mu = \frac{du^\mu}{d\tau},
\]

and “constant proper acceleration” means that the invariant magnitude is constant:

\[
-a^\mu a_\mu = a^2,
\]

where \(a>0\) is the proper acceleration measured in the instantaneous rest frame.

---

## 2. Parametrizing the four-velocity with rapidity

In \(1+1\) dimensions, it is convenient to write the four-velocity in terms of rapidity \(\theta(\tau)\):

\[
u^\mu = \left(c\cosh\theta,\; c\sinh\theta\right),
\]

so that indeed

\[
u^\mu u_\mu = c^2\cosh^2\theta - c^2\sinh^2\theta = c^2.
\]

Then

\[
a^\mu = \frac{du^\mu}{d\tau}
= c\dot\theta(\sinh\theta,\cosh\theta),
\]

where \(\dot\theta = d\theta/d\tau\). Its norm is

\[
a^\mu a_\mu
= c^2\dot\theta^2(\sinh^2\theta - \cosh^2\theta)
= -c^2\dot\theta^2.
\]

Therefore

\[
-a^\mu a_\mu = c^2\dot\theta^2 = a^2.
\]

So for constant proper acceleration,

\[
\dot\theta = \frac{a}{c},
\]

choosing the sign so acceleration is toward increasing \(x\). Hence

\[
\theta(\tau)=\frac{a\tau}{c} + \theta_0.
\]

We choose initial conditions so that at \(\tau=0\), the observer is instantaneously at rest, so \(\theta_0=0\). Then

\[
u^\mu = \left(c\cosh\frac{a\tau}{c},\; c\sinh\frac{a\tau}{c}\right).
\]

Thus

\[
\frac{dt}{d\tau} = \cosh\frac{a\tau}{c},\qquad
\frac{dx}{d\tau} = c\sinh\frac{a\tau}{c}.
\]

Integrating:

\[
t(\tau) = \frac{c}{a}\sinh\frac{a\tau}{c} + t_0,
\]

\[
x(\tau) = \frac{c^2}{a}\cosh\frac{a\tau}{c} + x_0.
\]

A standard convenient choice is

\[
t_0=0,\qquad x_0=0 \text{ shifted so that } x(0)=\frac{c^2}{a}.
\]

Then the worldline is

\[
ct(\tau) = \frac{c^2}{a}\sinh\frac{a\tau}{c},
\qquad
x(\tau) = \frac{c^2}{a}\cosh\frac{a\tau}{c}.
\]

---

## 3. Hyperbolic trajectory

Eliminate \(\tau\):

\[
x^2 - c^2 t^2
=
\left(\frac{c^2}{a}\right)^2
\left(\cosh^2\frac{a\tau}{c} - \sinh^2\frac{a\tau}{c}\right)
=
\left(\frac{c^2}{a}\right)^2.
\]

So the worldline is the hyperbola

\[
x^2 - c^2 t^2 = \frac{c^4}{a^2}.
\]

This is the standard uniformly accelerated trajectory.

---

## 4. From one hyperbola to a coordinate system: Rindler coordinates

The above hyperbola suggests foliating the **right Rindler wedge**

\[
x>|ct|
\]

by hyperbolae of constant \(x^2-c^2t^2\).

Define a spatial coordinate \(\xi>0\) by

\[
x^2-c^2 t^2 = \xi^2.
\]

Then points of fixed \(\xi\) lie on hyperbolae, and each hyperbola corresponds to a uniformly accelerated observer with proper acceleration

\[
a(\xi)=\frac{c^2}{\xi}.
\]

To parametrize motion along a given hyperbola, define a time coordinate \(\eta\) through

\[
ct = \xi \sinh\left(\frac{\eta}{c}\right),\qquad
x = \xi \cosh\left(\frac{\eta}{c}\right).
\]

This is one common normalization. Another common convention is to define dimensionless Rindler time \(\tilde\eta\) by

\[
ct = \xi \sinh \tilde\eta,\qquad x=\xi\cosh\tilde\eta.
\]

I will keep \(\eta\) with dimensions of length so that \(\eta/c\) is dimensionless.

Thus the coordinate transformation is

\[
\boxed{
x = \xi \cosh\left(\frac{\eta}{c}\right),\qquad
ct = \xi \sinh\left(\frac{\eta}{c}\right),\qquad \xi>0
}
\]

covering the right wedge \(x>|ct|\).

The inverse transformation is

\[
\boxed{
\xi = \sqrt{x^2-c^2 t^2},\qquad
\eta = c\,\mathrm{arctanh}\!\left(\frac{ct}{x}\right)
= \frac{c}{2}\ln\!\left(\frac{x+ct}{x-ct}\right)
}
\]

valid for \(x>|ct|\).

---

## 5. Metric in Rindler coordinates

Now compute the metric.

From

\[
x = \xi \cosh\left(\frac{\eta}{c}\right),\qquad
ct = \xi \sinh\left(\frac{\eta}{c}\right),
\]

we get

\[
dx = \cosh\left(\frac{\eta}{c}\right)d\xi
+ \frac{\xi}{c}\sinh\left(\frac{\eta}{c}\right)d\eta,
\]

\[
c\,dt = \sinh\left(\frac{\eta}{c}\right)d\xi
+ \frac{\xi}{c}\cosh\left(\frac{\eta}{c}\right)d\eta.
\]

Then

\[
ds^2 = c^2dt^2 - dx^2.
\]

Substituting and simplifying using \(\cosh^2-\sinh^2=1\), the cross terms cancel and one obtains

\[
ds^2
=
\left(\frac{\xi^2}{c^2}\right)d\eta^2 - d\xi^2.
\]

Including transverse directions \(y,z\),

\[
\boxed{
ds^2 = \left(\frac{\xi^2}{c^2}\right)d\eta^2 - d\xi^2 - dy^2 - dz^2.
}
\]

This is the Rindler metric in the right wedge.

---

## 6. Relation to proper time and proper acceleration

For an observer at fixed \(\xi=\xi_0\), \(dy=dz=0\),

\[
ds^2 = c^2 d\tau^2 = \left(\frac{\xi_0^2}{c^2}\right)d\eta^2.
\]

Hence

\[
d\tau = \frac{\xi_0}{c^2}d\eta.
\]

So \(\eta\) is proportional to the proper time along each constant-\(\xi\) worldline.

At fixed \(\xi_0\),

\[
ct = \xi_0 \sinh\left(\frac{\eta}{c}\right),\qquad
x = \xi_0 \cosh\left(\frac{\eta}{c}\right).
\]

Comparing with the uniformly accelerated trajectory

\[
ct = \frac{c^2}{a}\sinh\frac{a\tau}{c},\qquad
x = \frac{c^2}{a}\cosh\frac{a\tau}{c},
\]

we identify

\[
\xi_0 = \frac{c^2}{a}.
\]

Thus

\[
\boxed{
a = \frac{c^2}{\xi}.
}
\]

So each \(\xi=\mathrm{const}\) worldline is uniformly accelerated, with larger acceleration as \(\xi\to 0\).

---

## 7. The Rindler horizon

The coordinate patch only covers \(x>|ct|\). The boundaries are

\[
x=ct,\qquad x=-ct.
\]

These are null lines in Minkowski spacetime, because along them

\[
ds^2 = c^2dt^2-dx^2=0
\quad\text{when } dx=\pm c\,dt.
\]

So they are null surfaces (in \(1+1\), null lines).

For the right wedge observer, the relevant future/past horizons are these two null boundaries. In null coordinates

\[
u = ct-x,\qquad v = ct+x,
\]

the right wedge is characterized by

\[
u<0,\qquad v>0.
\]

The boundaries are

\[
u=0 \quad (x=ct),\qquad v=0 \quad (x=-ct).
\]

These are the Rindler horizons.

---

## 8. Why the horizon is a coordinate singularity in Rindler coordinates

From the inverse transformation,

\[
\xi = \sqrt{x^2-c^2t^2}.
\]

On the horizon \(x^2-c^2t^2=0\), so

\[
\xi=0.
\]

The metric is

\[
ds^2 = \left(\frac{\xi^2}{c^2}\right)d\eta^2 - d\xi^2.
\]

At \(\xi=0\), the coefficient \(g_{\eta\eta}\) vanishes. This looks singular in these coordinates. But Minkowski spacetime itself is perfectly regular there; the apparent singularity is only because the coordinates break down at the wedge boundary.

So:

- In Minkowski coordinates the horizon is just the regular null surface \(x=\pm ct\).
- In Rindler coordinates it appears at \(\xi=0\), where the chart degenerates.

Thus it is a **coordinate singularity**, not a curvature singularity. Indeed the spacetime is flat everywhere, with Riemann tensor zero.

# Part 2: Unruh temperature from quantum field theory

Now consider a massless scalar field \(\phi\) in Minkowski vacuum \(|0_M\rangle\).

I will state clearly what is assumed and what is derived.

---

## 1. Key physical assumption

The key assumption/result is:

> The field can be decomposed into positive-frequency modes with respect to Minkowski time \(t\), and also into positive-frequency modes with respect to Rindler time \(\eta\). These two mode decompositions are related by a Bogoliubov transformation, and the Minkowski vacuum is not the Rindler vacuum.

This requires standard QFT in curved spacetime methods. I will derive the thermal form in the usual way, but some steps invoke known results about analytic continuation and Bogoliubov coefficients rather than deriving every functional analysis detail from scratch.

So this part is partly **heuristic/standard** unless one does a full mode normalization calculation.

---

## 2. Scalar field equation and mode decompositions

A massless scalar field satisfies

\[
\Box \phi = 0.
\]

In \(1+1\) dimensions this is

\[
\left(\frac{1}{c^2}\partial_t^2 - \partial_x^2\right)\phi = 0.
\]

Introduce Minkowski null coordinates

\[
U = ct-x,\qquad V=ct+x.
\]

Then the general solution splits into left- and right-moving parts:

\[
\phi(t,x)=\phi_R(U)+\phi_L(V).
\]

It is enough to analyze one chiral sector, say the right-moving part depending on \(U\).

---

## 3. Rindler null coordinates

In the right wedge \(x>|ct|\), define Rindler null coordinates. From

\[
ct=\xi\sinh\left(\frac{\eta}{c}\right),\qquad
x=\xi\cosh\left(\frac{\eta}{c}\right),
\]

we have

\[
U = ct-x
= \xi\left(\sinh\frac{\eta}{c}-\cosh\frac{\eta}{c}\right)
= -\xi e^{-\eta/c},
\]

\[
V = ct+x
= \xi\left(\sinh\frac{\eta}{c}+\cosh\frac{\eta}{c}\right)
= \xi e^{\eta/c}.
\]

Thus in the right wedge,

\[
\boxed{
U=-\xi e^{-\eta/c}<0,\qquad
V=\xi e^{\eta/c}>0.
}
\]

A useful “Rindler null time” variable is

\[
u_R = \eta - c\ln(\xi/\xi_0),
\]

so that

\[
U = -\xi_0 e^{-u_R/c},
\]

for some arbitrary constant \(\xi_0\) inserted to make the logarithm dimensionless.

The crucial point is that Minkowski null coordinate \(U\) is **exponential** in Rindler null coordinate \(u_R\).

---

## 4. Positive frequency mismatch

A positive-frequency Minkowski right-moving mode has the form

\[
f_\Omega(U)\propto e^{-i\Omega U/c}
\]

with \(\Omega>0\), i.e. positive frequency with respect to inertial time.

A positive-frequency Rindler mode has the form

\[
g_\omega(u_R)\propto e^{-i\omega u_R/c}
\]

with \(\omega>0\), i.e. positive frequency with respect to Rindler time \(\eta\).

Since

\[
U = -\xi_0 e^{-u_R/c},
\]

a Minkowski mode restricted to the right wedge becomes

\[
f_\Omega \propto \exp\!\left(i\frac{\Omega \xi_0}{c}e^{-u_R/c}\right).
\]

This is not a pure Fourier mode in \(u_R\); it contains both positive and negative Rindler frequencies. That is the origin of the Bogoliubov mixing and hence particle perception by accelerated observers.

---

## 5. Bogoliubov transformation

Expand the field either in Minkowski modes or Rindler modes:

\[
\phi = \sum_i \left(a_i^M f_i + a_i^{M\dagger} f_i^*\right)
\]

or

\[
\phi = \sum_j \left(a_j^R g_j + a_j^{R\dagger} g_j^*\right).
\]

Then the mode sets are related by

\[
g_j = \sum_i \left(\alpha_{ji} f_i + \beta_{ji} f_i^*\right),
\]

and correspondingly

\[
a_j^R = \sum_i \left(\alpha_{ji}^* a_i^M - \beta_{ji}^* a_i^{M\dagger}\right).
\]

The Minkowski vacuum is defined by

\[
a_i^M |0_M\rangle = 0.
\]

The number of Rindler quanta in mode \(j\) seen in \(|0_M\rangle\) is

\[
\langle 0_M|\, a_j^{R\dagger} a_j^R \,|0_M\rangle
= \sum_i |\beta_{ji}|^2.
\]

So we must determine the ratio of Bogoliubov coefficients.

---

## 6. Thermal factor from analytic continuation

This is the central step. I will outline the standard derivation.

Because \(U=-\xi_0 e^{-u_R/c}\), a Rindler positive-frequency mode behaves like

\[
g_\omega \propto e^{-i\omega u_R/c}
\propto (-U)^{i\omega c^{-1}\cdot c}
\]

more carefully,

\[
u_R = -c\ln\!\left(\frac{-U}{\xi_0}\right),
\]

hence

\[
e^{-i\omega u_R/c}
=
\exp\!\left(i\omega \ln\!\frac{-U}{\xi_0}\right)
=
\left(\frac{-U}{\xi_0}\right)^{i\omega}.
\]

This expression is naturally defined in the right wedge where \(U<0\). To continue it across the horizon \(U=0\), one must choose a branch of the logarithm. The analyticity condition appropriate to positive Minkowski frequency leads to the relation between positive and negative Rindler frequency parts:

\[
|\beta_\omega|^2 = e^{-2\pi \omega c/a}\, |\alpha_\omega|^2.
\]

I am invoking here a standard result from the full analytic continuation/Bogoliubov calculation. The factor \(2\pi\) comes from circling the branch point in the complex \(U\)-plane.

Using the Bogoliubov identity for bosons,

\[
|\alpha_\omega|^2 - |\beta_\omega|^2 = 1,
\]

we get

\[
|\beta_\omega|^2
=
\frac{1}{e^{2\pi \omega c/a}-1}.
\]

This is exactly the Bose-Einstein occupation number at temperature \(T\) if

\[
\frac{\hbar\omega}{k_B T} = \frac{2\pi \omega c}{a}.
\]

Therefore,

\[
\boxed{
T = \frac{\hbar a}{2\pi c k_B}.
}
\]

This is the Unruh temperature.

---

## 7. More explicit identification of the acceleration scale

A detector moving on the worldline \(\xi=c^2/a\) has proper time \(\tau\) related to Rindler time \(\eta\) by

\[
d\tau = \frac{\xi}{c^2}d\eta = \frac{1}{a} d\eta.
\]

So frequencies conjugate to \(\eta\) are proportional to proper-time frequencies measured by that observer. The thermal factor can thus be written physically as

\[
\langle N_\omega\rangle
=
\frac{1}{e^{\hbar\omega/(k_B T)}-1}
\]

with the above \(T\).

---

## 8. Equivalent derivation via Euclidean periodicity

There is another standard derivation, elegant and compact, based on imaginary time. I include it because it makes the temperature especially transparent.

Start with the Rindler metric

\[
ds^2 = \left(\frac{a\rho}{c}\right)^2 c^2 d\tau_R^2 - d\rho^2 - dy^2 - dz^2,
\]

where one often writes coordinates so that

\[
ds^2 = a^2\rho^2 d\tau_R^2 - d\rho^2 - dy^2 - dz^2
\]

after suitable units/conventions. Wick rotate to Euclidean time:

\[
\tau_R \to -i\tau_E.
\]

Then near \(\rho=0\),

\[
ds_E^2 = d\rho^2 + \left(\frac{a\rho}{c}\right)^2 c^2 d\tau_E^2 + \cdots
= d\rho^2 + a^2\rho^2 d\tau_E^2 + \cdots
\]

which is just the flat plane in polar coordinates if \(\tau_E\) is an angular variable. To avoid a conical singularity at \(\rho=0\), one requires periodicity

\[
\tau_E \sim \tau_E + \frac{2\pi c}{a}.
\]

In finite-temperature QFT, imaginary time periodicity \(\beta\hbar\) corresponds to temperature \(T\), so

\[
\beta\hbar = \frac{2\pi c}{a}.
\]

Hence

\[
\beta = \frac{1}{k_B T} = \frac{2\pi c}{\hbar a}
\]

and therefore

\[
\boxed{
T = \frac{\hbar a}{2\pi c k_B}.
}
\]

This derivation is standard but also invokes known finite-temperature QFT machinery.

# Physical interpretation

A uniformly accelerated observer does not have access to the full Minkowski spacetime, only one Rindler wedge. The Minkowski vacuum is a globally pure state, but when restricted to one wedge, the inaccessible modes beyond the horizon are traced over. The resulting reduced state is thermal with respect to the boost generator, i.e. with respect to Rindler time evolution.

So:

- inertial observer: sees vacuum,
- uniformly accelerated observer: sees a thermal bath,

with temperature

\[
\boxed{T_U=\frac{\hbar a}{2\pi c k_B}}.
\]
